Alternative TOC

3rd January 2019 at 12:16pm
Further development

The current "classical" TOC macros seems a bit awkward for building the TOC index (see Generating the TOC index). Instead, the drag-and-drop mechanism recently extended for lists (see list-links-draggable macro) looks much more suitable for hierarchical TOC interactive building and therefore could be used for TOC index building with much more ease.

It would be great if somebody from the TW community would challenge the development of this alternative TOC tool.
