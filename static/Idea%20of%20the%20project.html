Idea of the project

3rd January 2019 at 12:18pm
Autonum project

The whole idea of the project is to equip TW with tools for auto indexing of various types of internal objects in the text with successive referencing to them.

Very roughly the overall procedure comprises the following steps:

  1. Generating the TOC index
  2. Generating the object type indexes
  3. Internal text object in wiki text
  4. Storing the object type indexes
  5. Object manipulation
